# RPGMakerRoguelikePlugin

RPGツクールMVで使用できるローグライクゲームのためのプラグインです。

未完成品ゆえに、現状すべてのプラグインを使用してもローグライクゲームとしては動作しません。
ただし、いくつかのプラグインについては単体でも使用できるため、
主にそれらのために公開しています。

詳細はブログ記事をご参照ください。

http://fspace.hatenablog.com/entry/2016/09/22/164953
